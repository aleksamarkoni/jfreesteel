Могући проблеми

#1 Програм приказује грешку да читач није препознат и затвара се

Проверите да ли је читач исправно инсталиран на систему тако што у терминалу покренете наредбу pcsctest. Наредба ће исписати информације о читачу укључујући и његов назив, а испис треба да „реагује“ када се картица убаци или извади из читача. Већина УСБ читача би требало да раде одмах по укључивању, али за неке може бити потребан pcscd драјвер (ifhandler) из складишта пакета или са сајта произвођача.

Програм би требало да захтева инсталацију Oracle Java радног окружења ако оно није инсталирано. Mac OS X 10.6 и 10.7 долазе са Јавом 1.6, али ту је у скрипт за покретање неопходно додати параметар -d32 како би Јава могла се повеже са читачем. Ако је одзив наредбе у терминалу исправан, претпостављајући да сте апликацију инсталирали у /Applications покушајте да у терминалу покренете: 

java -d32 -jar /Applications/JFreesteel\ eID\ Viewer.app/Contents/Java/eidviewer.jar

На Mac OS X 10.8 и новијем ова наредба ће захтевати инсталацију Oracle Java JDK развојног окружења. Уместо тога, ако сте инсталирали Oracle Java радно окружење (на пример са сајта java.com) покушајте да у терминалу извршите: 

/Library/Internet\ Plug-Ins/JavaAppletPlugin.plugin/Contents/Home/bin/java -d32 -jar /Applications/JFreesteel\ eID\ Viewer.app/Contents/Java/eidviewer.jar
Најбоље да наредбу означите и копирате у прозор терминала, пазите да исправно означите од почетка до краја. При куцању наредбе у терминалу код сваког дела наредбе можете да почнете са пар слова и онда притиснете тастер Tab. Наредбу треба унети у једном реду, у прозору терминала може да се сама преломи на крају прозора али немојте куцати тастер за нови ред!

Уколико ово помогне, јавите резултат е-поштом.

#2 Програм се приказује на енглеском језику

Програм ће се приказивати на српском уколико је српски подешен као подразумевани језик на систему. Отворите подешавања система и потражите листу у групи Language & Region или на старијим системима у групи Language & Text на првој картици Languages. Прво додајте српски у листу, а затим га превуците изнад енглеског. Може бити неопходно да рестартујете рачунар након што потврдите подешавања.

#3 Програм не приказује никакву грешку, али се уопте не покреће

Програм захтева инсталацију Oracle Java окружења. Због величине датотеке (преко 100MB) Јава окружење није укључено у пакет апликације. Требало би да уколико на систему није већ инсталирана Јава (због покретања Јава аплета на пример) програм затражи инсталацију при првом покретању. У сваком случају покушајте да са сајта java.com преузмете и инсталирате Oracle Java окружење.

#4 Приказује се грешка да није дозвољено покретање небезбедних програма

Apple Gatekeeper може да онемогући покретање програма који нису аутентификовани кроз Developer ID програм. Уколико кликнете десним тастером и из менија одаберете покретање апликације требало би да се прикаже прозор у коме можете да потврдите изузетак и наставите коришћење програма.

Gatekeeper подешавања се налазе у групи „Security & Privacy“. Apple Gatekeeper је користан и немојте га олако искључивати. Али уколико имате проблем са покретањем ове апликације, покушајте да привремено искључите, покушате да тада покренете програм и затим вратите подешавања на претходну вредност.

Уколико поседујете Apple Developer ID сертификат, и желите да помогнете и њиме потпишете издања програма JFreesteel eID Viewer, погледајте инструкције у Гит складишту и јавите се е-поштом.

#5 Сачувани ПДФ извештај садржи само слику

Програм тренутно не приказује остале податке у ПДФ извештају. Уколико познајете или тек учите Јаву, додавање ове функционалности у класу viewer/PdfReport.java може да буде забавно. Локације за редове текста у пикселима пронађите овде.

Уколико успете, допуну пошаљите е-поштом или кроз GitHub и функција ће се наћи у новом издању.
