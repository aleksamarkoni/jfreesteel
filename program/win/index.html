Снимак екрана програма

Могућности и познати недостаци:

  • Препознаје убачену личну карту и успешно чита податке и слику
  • Омогућава чување ПДФ извештаја, укључујући податке и слику
  • Не омогућава проверу електронског потписа података на личној карти
  • Не омогућава извоз личних сертификата са личне карте
  • При употреби већег системског фонта, приказ може бити исечен сдесна. Довољно је повећати прозор програма
JFreesteel eID Viewer 1.1
2,8 MB — 15.02.2015

Ово није званични Читач електронске личне карте (Челик+) који можете да преузмете са сајта сертификационог тела МУП-а. JFreesteel eID Viewer је независан слободан програм који уједно приказује рад библиотеке JFreesteel намењене програмерима који би да очитавања личне карте уграде у своје програме.

За разлику од званичног програма за читање, JFreesteel eID Viewer није само за Microsoft Windows већ ради и на другим системима као што су ГНУ/Линукс и Mac OS X. Ипак осим што је слободан софтвер па можете да „завирите“ у код и сазнате како тачно програм ради, на Windows-у овај програм нема предности у односу на званичан софтвер, а има пар функционалности које му недостају.

Упутство за инсталацију

Двокликните преузету датотеку и започните инсталацију
Кликните на дугме Install / Инсталирај
Кликните на дугме Finish / Заврши
Покрените програм двоклико на нову иконицу

Могући проблеми

#1 Програм приказује грешку да читач није препознат и затвара се

Проверите да ли је читач исправно инсталиран на систему кроз Device Manager у Control Panel-у. Уколико нисте раније користили читач, могуће да је потребно да инсталирате драјвере са сајта произвођача.

Системски сервис „Smart card service“ треба да буде омогућен. Овај сервис је подразумевано означен као омогућен и са аутоматским покретањем по потреби. Уколико програм не препознаје читач, покушајте да ручно покренете овај сервис кроз Administrative Tools > Services поставке у Control Panel-у.

#2 Програм се приказује на енглеском језику

Програм ће се приказивати на српском уколико је српски подешен на систему као подразумевани језик за приказ. Језик за приказ можете да промените тако што инсталирате српски језички пакет (LIP).

#3 Програм не приказује никакву грешку, али се уопте не покреће

Програм захтева инсталацију Oracle Java окружења. Због величине датотеке (преко 100MB) Јава окружење није укључено у пакет апликације. Требало би да уколико на систему није већ инсталирана Јава (због покретања Јава аплета на пример) програм затражи инсталацију при првом покретању. У сваком случају покушајте да са сајта java.com преузмете и инсталирате Oracle Java окружење.
