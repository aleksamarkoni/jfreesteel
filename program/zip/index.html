Снимак екрана програма

Могућности и познати недостаци:

  • Препознаје убачену личну карту и успешно чита податке и слику
  • Омогућава чување ПДФ извештаја, али у извештај уписује само слику, не и друге податке
  • Не омогућава проверу електронског потписа података на личној карти
  • Не омогућава извоз личних сертификата са личне карте
  • При употреби већег системског фонта, приказ може бити исечен сдесна. Довољно је повећати прозор програма
JFreesteel eID Viewer 1.0
2,3 MB — 24.09.2014

Упутство за инсталацију

  1. Подесите читач паметних картица и PC/SC компоненту на систему
  2. Инсталирајте Јава радно окружење (Oracle Java или OpenJDK JRE, издање 6 или новије).
    Можете да користите и другу варијанту све док је подржан Java Smart Card I/O API (JSR-268).
  3. Проверите да су Јава радно окружење и PC/SC компонента исправно повезани
  4. Распакујте преузету архиву
  5. Уколико ваш систем омогућава покретање JAR архива двокликните на eidviewer.jar. У противном на Униксоликим системима покрените шел-скрипту eidviewer која претпоставља да је java наредба доступна у стандардној путањи.

    Уколико не можете да покренете шел-скрипт или се java наредба налази на нестандардној путањи, у терминалу пређите у локацију где сте распаковали архиву и покрените програм наредбом: java -jar ./eidviewer.jar односно сличном, додајући одговарајућу путању до java наредбе.

Могући проблеми

#1 Програм приказује грешку да читач није препознат и затвара се

Проверите да ли је читач исправно инсталиран на систему и да ли Java Smart Card I/O API остварује комуникацију са системском PC/SC компонентом.

Упутства за решавање зависе од оперативног система.

На ГНУ/Линуксу систем користи pcscd из pcsc-lite, а читач захтева и посебан pcsc-lite драјвер (ifhandler). Већина УСБ читача подржавају CCID протокол и раде са libccid драјвером из pcsc-lite пројекта. Потражите и инсталирајте пакете pcsc-lite и libccid за вашу дистрибуцију, односно додатне ifhandler драјвере из складишта дистрибуције или са сајта произвођача читача. Можете да користите pcsc_scan из пројекта pcsc-tools да проверите да ли је читач коректно инсталиран. Програм у терминалу исписује информације о читачима, а требало би да „реагује“ када се картица убаци или извади из читача.

Повезивање pcscd и Јаве се на ГНУ/Линуксу реализује преко библиотеке libpcsclite. Без обзира коју дистрибуцију користите, прочитајте како да исправно успоставите везу између Јаве и ове библиотеке у одељку за Убунту ГНУ/Линукс дистрибуцију.

За информације како да проблем решите на системима Apple Mac OS X и Microsoft Windows погледајте посебне странице на сајту.

#2 Програм се приказује на енглеском језику

Програм ће се приказивати на српском уколико је активан српски локалитет.

На ГНУ/Линуксу поставите LANG променљиву на sr_RS.

#3 Сачувани ПДФ извештај садржи само слику

Програм тренутно не приказује остале податке у ПДФ извештају. Уколико познајете или тек учите Јаву, додавање ове функционалности у класу viewer/PdfReport.java може да буде забавно. Локације за редове текста у пикселима пронађите овде.

Уколико успете, допуну пошаљите е-поштом или кроз GitHub и функција ће се наћи у новом издању.
